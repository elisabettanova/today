# Today

Today is a software that helps you track your daily agile activities.

## Installation

**TL;DR command list**

    git clone git@github.com:midorikocak/today
    cd today
    composer install
    cp .env.example .env
    php artisan key:generate
    php artisan migrate

## Usage

Go to server.

## Testing

  phpunit
